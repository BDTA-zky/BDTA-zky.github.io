蓝桥杯--成绩统计（python）

发表于2023-04-27|更新于2023-05-06|python蓝桥杯
|字数总计:762|阅读时长:2分钟|阅读量:

题目描述

小蓝给学生们组织了一场考试，卷面总分为 100 分，每个学生的得分都是一个 0 到 100 的整数。

如果得分至少是 60 分，则称为及格。如果得分至少为 85 分，则称为优秀。

请计算及格率和优秀率，用百分数表示，百分号前的部分四舍五入保留整 数。

输入描述

输入的第一行包含一个整数 n (1≤n≤10^4)，表示考试人数。

接下来 n 行，每行包含一个 0 至 100 的整数，表示一个学生的得分。

输出描述

输出两行，每行一个百分数，分别表示及格率和优秀率。百分号前的部分 四舍五入保留整数。

思路：按题目要求即输入人数和对应的成绩，并筛选出符合条件的人数占的百分比。

python中input()会将输入的语句当作字符串类型的，为了方便后面的使用，我们将其转换为int类型；下来要输入成绩，我们可以找一个地方来存放成绩，我这里是通过for循环将成绩存入cj_list列表中；成绩知道后，下来就是找符合条件的成绩个数，这里还是用到for循环来遍历cj_list列表，并通过if判断语句来找符合条件的（因为是通过input向列表中存入数据，那么存入的都是str类型，需要将其转换为int类型以方便比较）；题目要求的是输出百分比，并且要四舍五入，那就需要用到round函数，可以通过用符合条件的人数除以总人数再扩大百倍然后取整加一个%即可。

round函数是一个可以对数字四舍五入的函数。round（数字，参数）数字代表的是我们需要四舍五入的数，参数代表的是我们需要对哪个部分四舍五入；参数可正可负可为零，但注意需为整数，当为0时，代表的是对整数部分四舍五入；

为正整数时，代表的是对小数对应的位数四舍五入；

为负整数时，代表的是对整数部分对应的位数四舍五入。
