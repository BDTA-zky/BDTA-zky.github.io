库的下载方式：

可以win+R打开运行框，输入cmd，打开命令行，然后输入pip install 加包名

如：pip install pyttsx3

要注意一点就是下载速度比较慢，想提高下载速度可以先添加镜像源pip清华源（永久使用）：

pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
然后再重复下载方式即可
