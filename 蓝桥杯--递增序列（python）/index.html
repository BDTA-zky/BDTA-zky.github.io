解题思路：

本题的主要是比较每一行，每一列，以及每一个45°角上的每两个字母的大小，并找到输出按递增的序列。

首先是将其赋予一个字符串，并且要将其每一行分隔开来作为一个单独的小字符串。这里要用到split函数，其作用就是将字符串按照特定的字符分割成一个新的字符串

然后就要比较大小了，首先是遍历每一个字母，先行后列，就需要用到两个for循环，外层循环代表的是列，内层循环代表的是行。它所对应的就是每一行每一列的每一个字母

它有三种比较方式即按行比较，按列比较，按45°对角线比较

按行比较：在内层循环的基础上再加一层for循环，它的目的也是遍历每一个字母，带区别在于它的范围是基于第二层循环的，即当第二层循环进行一次，即为第一个字母的时候，第三层循环就从第一个字母开始向后遍历，遍历的每个字母依次与第一个字母比较，如果符合条件就令sum加一；当第一行都与第一个字母比较完之后，第二层循环进行第二次，即为第二个字母，而第三层循环从第二个字母开始遍历，并与第二个字母进行比较……当第一行全部字母都比较完了之后，第一层循环开始向下进行一次，然后第二层循环重复上述操作……一直到全部遍历完成

按列比较：原理与按行比较相同，不过就是第三层循环的范围变成了跟着第一层循环变得了，其比较也是一列一列进行的即上下比较

按45°对角线比较：拥有四层for循环，前两层不变，后两层循环与前两层循环相似，都是遍历每一个字母，最主要的是第四层里面的if判断语句。45°对角线有两种，一种是左下到右上，一种是右下到左上的。左下到右上的坐标里面，横坐标相减加上纵坐标相减的值为0且两个横坐标不相等

右下到左上则为，横坐标相减等于纵坐标相减，且横坐标相减大于0

在这里面还用到一个函数ord（）函数，其作用是将字符转换为对应的Ascll码值，方便比较大小
