排序（python）

发表于2023-05-05|更新于2023-09-26|python蓝桥杯真题2020年省赛
|字数总计:1.1k|阅读时长:3分钟|阅读量:

题目描述

本题为填空题，只需要算出结果后，在代码中使用输出语句将所填结果输出即可。

小蓝最近学习了一些排序算法，其中冒泡排序让他印象深刻。

在冒泡排序中，每次只能交换相邻的两个元素。

小蓝发现，如果对一个字符串中的字符排序，只允许交换相邻的两个字符， 则在所有可能的排序方案中，冒泡排序的总交换次数是最少的。

例如，对于字符串lan 排序，只需要 1 次交换。对于字符串qiao 排序，总共需要 4 次交换。

小蓝找到了很多字符串试图排序，他恰巧碰到一个字符串，需要 100 次交 换，可是他忘了吧这个字符串记下来，现在找不到了。

请帮助小蓝找一个只包含小写英文字母且没有字母重复出现的字符串，对 该串的字符排序，正好需 100 次交换。如果可能找到多个，请告诉小蓝最短的那个。如果最短的仍然有多个，请告诉小蓝字典序最小的那个。

解题思路：

本题的要求是找到符合冒泡排序次数的对应单词序列，而且是字典序最小的

字典序是指从前到后比较两个字符串的大小的方法。例如：ab abc ac是按照字典序从小到大排列。首先比较第一个字符，如果不同则第一个字符较小的字符串更小，如果相同则继续比较第2个字符……如此继续，来比较整个字符串的大小。

当逆序的时候字典序且字母靠前的是单词最小的。

那么可以将26个英文字母赋给一个变量，通过len()函数得到这个字符串的长度；用for循环开始遍历长度，用全逆乱序的冒泡排序次数找到符合条件的个数

全逆乱序的冒泡排序次数为 i(i-1)/2 i为需要排序的个数

用if找出最接近目标次数且比目标次数大的数，并算出比目标次数大多少，然后跳出循环；

将找到的个数即s=i值，其所包含的单词序列找出并逆序排列（要注意s所代表的是个数而不是下标，这里需要将s值减一，才是所需下标）；

将序列某部分找出并倒序排列所用的为**[n::-1]**; -1代表的是逆序全部排列，n代表的是从下标为0到下标为n的所有序列；这个整体代表的是将下标为0到下标为n的所有序列逆序排列。

写到这里这道题基本完成了，只差最后一步：就是将本序列的位置调动一下使冒泡排序次数达到目标要求；冒泡排序是两个两个的排序，我们可以根据多余的次数将一个单词比较的过程给抵消了，即将一个单词的顺序改一下让它少比较几次。

本题我们已经得知多余的次数，可以找的多余的次数作为下标对应的字母，将其的位置放到第一位，其他单词不变。这里我们会用到**[n:],[:m]**

[:m]代表的是取出下标为0到下标为m-1的序列

[n:]代表的是取出从下标为n到最后的序列
